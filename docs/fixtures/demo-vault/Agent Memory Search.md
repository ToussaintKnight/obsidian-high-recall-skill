# Agent Memory Search

This fixture note explains local-first AI search over private Obsidian notes. It focuses on high-recall retrieval for agents, research memory, private knowledge bases, local indexes, and workflows where missing relevant notes is worse than returning extra noise.

Relevant phrases: local-first, Obsidian notes, AI search, agent memory, private vault, high recall, research memory, recall pack.

The public benchmark fixture is not meant to prove quality on real vaults. It only makes install and evaluator behavior reproducible.
