# v0.1.0

First public release focused on making the project installable, testable, and safe to evaluate without a private vault.

Project page:

https://toussaintknight.github.io/obsidian-high-recall-skill/

Repository:

https://github.com/ToussaintKnight/obsidian-high-recall-skill

## What Is Included

- Local-first high-recall search for Obsidian vaults from Codex or CLI.
- Smart Connections `.smart-env` vector reuse when available.
- Local lexical fallback for fixture and no-vector cases.
- `obsidian-hybrid-search` fallback for hybrid/fulltext retrieval.
- Evaluator support for `smart`, `ohs`, and evaluator-derived `rrf-union`.
- Public fixture vault and smoke benchmark.
- CI, issue templates, PR template, security policy, contribution guide, and roadmap.
- Privacy-safe demo GIF generated from public fixture data.

## Try It

```bash
git clone https://github.com/ToussaintKnight/obsidian-high-recall-skill.git
cd obsidian-high-recall-skill
npm test
```

Query the public fixture vault:

```bash
node skills/obsidian-high-recall/scripts/obsidian_high_recall.mjs query "data collection for embodied AI robot demonstrations" --vault docs/fixtures/demo-vault --backend smart --limit 10
```

Use your own vault:

```bash
node skills/obsidian-high-recall/scripts/obsidian_high_recall.mjs query "your broad research query" --vault "/absolute/path/to/your-vault" --backend auto --limit 120
```

## Privacy Notes

The project does not intentionally upload vault contents. Do not include private note paths, snippets, queries, labels, `raw_runs.json`, or local cases files in issues or pull requests. See SECURITY.md:

https://github.com/ToussaintKnight/obsidian-high-recall-skill/blob/main/SECURITY.md

## Limitations

- The published benchmark is from a private, domain-skewed vault and should be treated as a deployment smoke test, not a universal benchmark.
- OHS fallback can be slow on first run because it may install packages, build a local index, or download local models.
- The npm package is GitHub-backed for now; external package registry publication is a later roadmap item.
